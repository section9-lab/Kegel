# KegelTimer

**macOS 用ケーゲル運動タイマー · タイムリーマインド · プライバシー優先**

[中文](README.md) | [English](README_EN.md)

![KegelTimer スクリーンショット](assets/screenshot.png)

## はじめに

KegelTimer は、スケジュール通りにケーゲル運動を行う習慣を作るための、ミニマルな macOS メニューバーアプリです。

- ⏰ **カウントダウンリマインダー**: 間隔時間を設定、自動でポップアップ通知
- 🌻 **フラワーアニメーション**: 収縮時に開花、リラックス時に閉じる
- 📊 **進捗トラッキング**: セグメント付きプログレスバーでトレーニング状況を可視化
- 🔒 **プライバシー優先**: アカウント不要、データ収集なし、完全ローカル動作

## クイックスタート

### 1. ダウンロードとインストール

```bash
# GitHub Releases からダウンロード
https://github.com/marswaveai/KegelTimer/releases/latest
```

`KegelTimer.app.zip` をダウンロード → 解凍 → `/Applications` フォルダにドラッグ

### 2. 初回使用

1. KegelTimer を開く（メニューバーに 🍑 アイコンが表示されます）
2. アイコンをクリック → **設定...** → リマインダー間隔を調整（デフォルト：60 分）
3. カウントダウン終了を待つ、または手動で **トレーニング開始** をクリック

### 3. トレーニングフロー

```
カウントダウン終了 → カードがポップアップ + ビープ音 → 
フラワーアニメーションに従う（収縮 → リラックス）→ 
すべてのセットを完了 → ×をクリックして閉じる → カウントダウン再開
```

## 機能

### メインインターフェース

| 状態 | 表示内容 | 操作 |
|------|----------|------|
| アイドル | 次のリマインダーまで：X 分 | カードをクリックして開始 |
| トレーニング中 | フラワーアニメーション + プログレスバー | カードをクリックして一時停止/再開 |
| 完了 | ✅ トレーニング完了！ | 自動閉じる、または×をクリック |

### フラワーアニメーション

- **開花**（収縮）: X 秒間キープ（設定可能：1-10 秒）
- **閉じる**（リラクゼーション）: X 秒間キープ（設定可能：1-10 秒）

### プログレスバー

- 各セグメントが 1 回のレップを表す
- 緑色 = 完了、灰色 = 未完了
- 下部に現在のセット/総セット数を表示

## 設定

**設定...**（ショートカット `⌘,`）を開いて以下を調整：

| オプション | 範囲 | デフォルト | 説明 |
|------------|------|------------|------|
| **リマインダー間隔** | 5-240 分 | 60 分 | リマインダーの頻度 |
| **収縮時間** | 1-10 秒 | 5 秒 | 収縮の保持時間 |
| **リラクゼーション時間** | 1-10 秒 | 5 秒 | レップ間の休憩時間 |
| **レップ数** | 1-30 レップ | 10 レップ | 1 セットのレップ数 |
| **セット数** | 1-10 セット | 3 セット | 1 セッションのセット数 |
| **セット間休憩** | 10-180 秒 | 60 秒 | セット間の休憩時間 |

### 設定例

**初心者モード**（約 3 分）:
- 収縮 3 秒、リラクゼーション 3 秒
- 1 セット 5 レップ、2 セット
- セット間休憩 30 秒

**上級者モード**（約 10 分）:
- 収縮 10 秒、リラクゼーション 5 秒
- 1 セット 15 レップ、5 セット
- セット間休憩 60 秒

## よくある質問

### Q: ケーゲル運動とは何ですか？

A: ケーゲル運動は骨盤底筋を強化する運動で、膀胱制御やコアの安定性の向上に役立ちます。新しい運動ルーチンを始める前に、医療提供者に相談することをお勧めします。

### Q: リマインダーを解除するにはどうすればいいですか？

A: カードの右上にある×ボタンをクリックして閉じ、カウントダウンをリセットします。または、メニューバーのアイコンをクリック → トレーニングを停止します。

### Q: データは収集されますか？

A: **いいえ**。すべてのデータはローカルの UserDefaults に保存され、サーバーにアップロードされることはありません。

### Q: Apple Silicon に対応していますか？

A: はい。Swift でネイティブ開発されており、Intel および Apple Silicon Mac の両方をサポートしています。

### Q: 最小システム要件は？

A: macOS 14.0 (Sonoma) 以降。
